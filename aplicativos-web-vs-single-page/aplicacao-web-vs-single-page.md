## Aplicação web tradicional vs Single page application

Traditional web application 

- Cliente (navegador) -> fazendo pedido (https://app.rocketseat.com.br) -> servidor (DB) -> recebe uma resposta (response) algo fixo (html,css,js)

Single page application
- Cliente (navegador) -> fazendo pedido (https://rocketseat.com.br/api/v1) -> servidor (db) resposta (response) -> devolve um json e o front atualiza so aquela parte que "mudou"
