## Site estático vs Site dinâmico

- Site estático
  Cliente faz um request (https://rockeatseat.com.br) vai para o servidor da rockeatseat e tem uma resposta (response) vem uma página (html,css,js) e não mudam ou seja o servidor não voltou nada a mais

- Site dinâmico
  Cliente faz um request (https://app.rockeatseat.com.br/me/matheus) por ter passado (/me/matheus) vai para o servidor e nesse caso o servidor ira buscar a informação no banco de dados e contruir uma pagina adptada conforme o pedido da pessoa.
